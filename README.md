# Hotel Room Service AI Agent

A multi-language voice agent for hotel room service, designed to be "better than human".

## Tech Stack
- **Cloud Hosting:** Render
- **Version Control:** GitHub
- **Telephony:** Twilio Programmable Voice
- **AI/Logic:** (To be determined: Python/FastAPI or Node.js) with OpenAI/Anthropic for LLM capabilities.

## Features
- Handles voice calls from hotel guests.
- Multi-language support (automatically detects and speaks guest's language).
- Integrates with hotel services (simulated for now).
- "Better than human" interactions: empathy, context awareness, instant knowledge.

## Setup
1. Configure Twilio Webhook to point to the Render service.
2. Set up environment variables (TWILIO_ACCOUNT_SID, OPENAI_API_KEY, etc.).
